### How to install

Clone repo:

`git clone git@github.com:Jordomav/pyqt.git`

Set up virtual env:

`python3 -m venv venv`

Start the virtual env:

`source venv/bin/activate`

Install deps:

`pip install fbs PyQt5==5.9.2`


### Start app

`fbs run`


### Create executable

`fbs freeze`

### Create installer

`fbs installler`
